## 自建的 BT 资源分享网

这是一个BT资源分享的网站，用户可分享自己的一些BT资源链接，亦可评论
项目原型是国外的一家BT分享网 (http://btsynckeys.com/) 

* 我在此项目中负责前端开发，项目应用 Jquery 构建页面文件
- 并用 PHP 构建基本的后台数据处理文件
* 后台使用 WampServer 的 php + mySQL 的集成环境
